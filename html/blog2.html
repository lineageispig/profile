Qt GUI

Learning C++ concept with Qt GUI

Overview: I always find the easiest way to learn coding, and I think the best way to learn how to code c++ is to use Qt Graphic User Interface.




Let's create a new project:

We open a new project in Qt IDE




The second step is to select all kits, including desktop and ios devices.




The third step is to define the base class. We select QMainWindow.




Click Done to create the project.




The project include Header file, Source file, and Forms for UI design.




Directly to our UI design page.




We click on the mainwindow.ui file, and then we add a pushbutton on our UI




Click and Drag the button to the user interface.




We type the following code to our mainwindow.cpp file 

ui->pushButton->setText("Hello"); // press the push button and it will
      //lead you to the design page




We run our project and we can get our first button at the UI!
