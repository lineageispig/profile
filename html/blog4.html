Opencv installation and code testing

The first try to use opencv to build project in xcode.

本文章在介紹如何安裝OpeCV 於 mac 上面，並且使用xcode 建置第一個Project.

大綱:




簡單製作一個OpenCV 安裝於mac的教學。安裝起來還是滿複雜的, 需要用到Unix，所以我做一個安裝教學，依照下面的步驟去執行，就可以建置一個opencv 的方案。

mac 版本：macOS Sierra 10.12:

我使用的mac為2012年的 MacBook Pro 13




下載最新的版本opencv: 3.2.0。下載opencv  http://opencv.org/




安裝cmake: type the following code on the command line:




for mac 在搜尋輸入terminal 開啟Unix 並且輸入以下的code 安裝hombrew 指令。 

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/
        go/install)"




之後會用指令來安裝cmake。 輸入：brew install cmake

brew install cmake




安裝 xcode 開啟App 下載xcode。




安裝 opencv: cd進入資料夾 輸入cd 指令進下載好的opencv 資料夾 cd desktop/opencv2.framework




輸入以下的code 建置一個release 資料夾於opencv資料夾裡。 

mkdir release
      cd release
      cmake -G "Unix Makefiles" ..
      make




開始在xcode create a project. 建立opencv project: 開啟xcode 建立專案： 建立command line tool 專案。




building setting 中 search path 建立路徑：

/usr/local/lib




header search paths 中建立  

/usr/local/include   /usr/local/include/opencv




測試部分：在main.cpp 檔案複製並且覆蓋原code. 並且在圖片路徑中輸入自己電腦中的圖片路徑。成功跑出來就好嘍，這樣就完成實作的部分！

//  Created by vince on 2013.
      //  Copyright (c) 2013年 vince. All rights reserved.
      #include "opencv2/core/core.hpp"
      #include "opencv2/highgui/highgui.hpp"
      #include "opencv2/imgproc/imgproc.hpp"


      using namespace cv;
      using namespace std;

      int main(int argc, char *argv[])
      {
      cv::Mat src;
    
      src=cv::imread("/Users/mac/Documents/MyCode/biglena.bmp", -1);

      //上面的路徑為圖片路徑

      cv::imshow("show image",src);  //這邊會show圖出來

      cv::waitKey(0);

      return 0;
      }
